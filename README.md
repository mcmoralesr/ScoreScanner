# ScoreScanner
Este es el repositorio para ScoreScanner, una aplicación híbrida que reconoce partituras manuscritas y las convierte a un formato MusicXML para su posterior edición en programas editores de partituras.

## Getting Started
### Creado con
* Python 3.6
* Tensorflow
